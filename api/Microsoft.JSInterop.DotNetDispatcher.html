ReleaseDotNetObject(Int64)

Releases the reference to the specified .NET object. This allows the .NET runtime to garbage collect that object if there are no other references to it.

To avoid leaking memory, the JavaScript side code must call this for every .NET object it obtains a reference to. The exception is if that object is used for the entire lifetime of a given user's session, in which case it is released automatically when the JavaScript runtime is disposed.
