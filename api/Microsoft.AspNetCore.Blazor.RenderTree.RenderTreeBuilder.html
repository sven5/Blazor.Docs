Remarks

This method is provided for infrastructure purposes, and is used to be UIEventArgsRenderTreeBuilderExtensions to provide support for delegates of specific types. For a good programming experience when using a custom delegate type, define an extension method similar to AddAttribute(RenderTreeBuilder, Int32, String, Action<UIChangeEventArgs>) that calls this method.
